# CODEX AI Hackathon 🚀

This is the official web platform for the **CODEX AI Hackathon**, a premier development marathon designed for visionaries to bring AI-powered solutions to life. This website serves as a central hub for detailed event information, featuring fully integrated application and communication workflows.

## 🎯 Mission Statement

The **CODEX AI Hackathon** is organized through a strategic collaboration between the **Turkish-German University InformatiX Club** and the **Ministry of Youth and Sports**. 

Our mission is to create a rich ecosystem where innovative minds can keep pace with global technological advancements without being bound by traditional limits. We aim to:
- Provide a platform for teams to develop and present **AI-powered solutions** to contemporary, real-world problems.
- Facilitate high-level **know-how exchange** between participants, industry professionals, and academics.
- Bridge the gap between innovative project teams and the **investor ecosystem**, turning prototypes into scalable ventures.

## 🛠️ Tech Stack

Built with a focus on elegance and performance, the site utilizes a clean, modern stack:
- **HTML5 & CSS3:** Structured with semantic tags and styled with the **Catppuccin Mocha** palette for a premium developer-centric aesthetic.
- **Vanilla JavaScript:** Powers smooth scroll-reveal animations and event-driven UI logic.
- **Canvas API:** An optimized, custom particle system for the animated hero background.
- **Google Fonts:** Utilizing **Inter** for professional, high-readability typography.
